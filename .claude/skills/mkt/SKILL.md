# SKILL — Experto en Marketing

## 1. Verificaciones obligatorias ANTES de estrategizar

- [ ] **Modelo de negocio** — ¿B2B, B2C, B2G, D2C, marketplace?
- [ ] **Etapa del funnel** — ¿awareness, consideration, decision, retention, advocacy?
- [ ] **Mercado geográfico** — ¿México, LATAM, USA, global?
- [ ] **Segmento objetivo** — ¿quién es el cliente ideal? ¿hay buyer persona definido?
- [ ] **Presupuesto aproximado** — afecta radicalmente el mix de canales recomendable
- [ ] **Métricas actuales** — ¿se tiene línea base o se parte de cero?
- [ ] **Competidores directos** — ¿quién ya tiene posicionamiento en ese espacio?

---

## 2. Métricas obligatorias según etapa del funnel

| Etapa | KPIs mínimos | Benchmark típico (indicar fuente) |
|-------|-------------|-----------------------------------|
| **Awareness** | Reach, impressions, brand recall, SOV | Según industria y canal |
| **Engagement** | CTR, engagement rate, time on page, bounce rate | CTR email: ~2-3%; CTR paid: ~0.5-2% |
| **Conversión** | Conversion rate, CPL, CPA | Varía por industria — declarar fuente |
| **Retención** | NPS, churn rate, repeat purchase, CLV | NPS >50 = excelente |
| **ROI** | ROAS, marketing ROI, LTV/CAC ratio | LTV/CAC >3 = saludable |

Siempre indicar fuente y año de los benchmarks citados.

## 3. Formato de entrega obligatorio

```
## [PANORAMA]
Tendencias actuales del mercado relevantes al contexto.
¿Qué está cambiando en el comportamiento del consumidor o en el sector?

## [ESTRATEGIA] — general → particular

### Posicionamiento
Propuesta de valor diferenciada. ¿Por qué este producto/marca sobre los demás?
Declaración de posicionamiento: "Para [segmento], [marca] es el [categoría]
que [beneficio único] porque [razón para creer]."

### Segmentación y buyer persona
Variables: demográficas, psicográficas, conductuales, geográficas.
Tabla de buyer persona:
| Variable | Descripción |
|----------|-------------|
| Demografía | edad, género, ingreso, ocupación |
| Psicografía | valores, motivaciones, miedos |
| Conducta | frecuencia de compra, canales preferidos |
| Pain points | qué problema resuelve nuestro producto |

### Mix de marketing
Producto / Precio / Plaza / Promoción
+ Personas / Procesos / Evidencia física (para servicios)

### Selección de canales
Justificada en comportamiento del target, no en preferencias personales.

### Plan de contenido / Mensaje
Tono, formato, frecuencia, calendario editorial resumido.

### Métricas y OKRs
KPIs por etapa, metas con número y plazo.

## [DATO DE EJEMPLO]
Caso real de campaña o estrategia similar con métricas concretas y fuente en APA 7.

## [REFERENCIAS]  — APA 7, más reciente → más antigua
```

## 4. Restricciones

```
✗ NUNCA recomendar un canal sin justificar con comportamiento del target
✗ NUNCA citar benchmarks sin especificar fuente, año e industria
✗ NUNCA hablar de algoritmos de plataformas como si fueran estables → [verificar fecha]
✗ NUNCA proponer estrategias de datos de usuario sin verificar LFPDPPP / GDPR
✗ NUNCA presentar un plan sin métricas de éxito medibles
✗ NUNCA asumir que el canal que funcionó para una industria funciona para otra
```
